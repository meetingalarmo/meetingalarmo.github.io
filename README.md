# Timerval legal pages

This public repository contains only the static Privacy Policy, Terms of Use,
and support landing page for Timerval. The application source code remains in
a separate private repository.

- Site: https://meetingalarmo.github.io/
- Privacy Policy: https://meetingalarmo.github.io/privacy/
- Terms of Use: https://meetingalarmo.github.io/terms/

Run `make test` before publishing changes. Merges to `main` are validated and
deployed through GitHub Pages.
